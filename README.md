dots-emulator
=============

A server emulator for the game Click Upon Dots by Jiggmin.

Usage
=====
    cd server
    npm install
    node app.js

Edit your `/etc/hosts` file and add the line

    127.0.0.1  jiggmin2.com

Run the included .swf file, or any other Click Upon Dots game client.
